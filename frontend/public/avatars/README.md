# Avatar Images Directory

This directory contains the user profile avatar images.

## Setup Instructions

1. Add your 12 avatar images here with the following names:
   - avatar1.png (or .jpg, .webp)
   - avatar2.png
   - avatar3.png
   - ... through avatar12.png

2. Image specifications:
   - Size: 200x200px (recommended, square format)
   - Format: PNG, JPG, or WebP
   - File size: Keep under 100KB each

3. Once you add the images, they will automatically appear in the avatar selector on the profile page.

## Notes
- All 12 images are optional - add as many as you want
- The avatar selector will display all images found in this directory
- Images are cached by the browser, so clear browser cache if you update images
- Use square dimensions for best appearance

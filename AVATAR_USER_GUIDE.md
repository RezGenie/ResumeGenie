## Settings

The avatar selector is available in the Profile Information section when in **Edit Mode** only.

**Edit Mode**: Click "Edit" button to:
1. Select a new avatar
2. Update name, phone, location, bio
3. Adjust job preferences

**View Mode**: Click "Cancel" or "Save" to exit edit mode
